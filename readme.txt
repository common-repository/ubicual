== Description ==

Este plugin permite recoger los datos personales de sus usuarios y guardarlos en una lista de ubicual.com

Puede crear su widget en ubicual.com.

== Installation ==

1. Suba o instale el plugin en su carpeta de plugins
2. Vaya a Apariencia > Widgets y seleccione ubicual
3. Posicione el widget en el lugar deseado
4. Rellene los campos del widget.

== Frequently Asked Questions ==

Como creo una widget?

Entre en https://ubicual.com, regístrese y podrá crear una landing page.
